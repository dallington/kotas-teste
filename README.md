# Kotas - Teste de Frontend

### Projeto utilizando Vue 3

## Project setup

```
yarn install
npm install
```

### Compiles and hot-reloads for development

```
yarn dev
npm run dev
```

### Serve Preview

```
yarn serve
npm run serve
```

### Compiles and minifies for production

```
yarn build
npm run build
```

### Lints and fixes files

```
yarn lint
npm run lint
```
